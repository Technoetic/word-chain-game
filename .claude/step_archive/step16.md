# Step 16 - HTML 컴포넌트화

모듈화된 JavaScript 구조에 맞춰 HTML을 **최대한** 재사용 가능한 컴포넌트로 분리한다.

## 파일 구조 규칙

- HTML 엔트리는 `src/index.html`에 작성
- JS/CSS는 외부 파일로 참조 (`<script src="js/...">`, `<link href="css/...">`)
- **인라인 스크립트/스타일 삽입 금지** — Step 36에서 html-bundler.ps1이 자동 번들링

합리적인 선에서 최대한 많은 서브에이전트를 병렬로 사용하여 컴포넌트화를 수행한다.

**컴포넌트화 단계에서 절대로 plan mode를 사용하지 않는다.**

서브에이전트는 항상 haiku를 사용한다.
