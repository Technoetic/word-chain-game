# Step 33 - 최적화 기획

step32_최적화조사_chunk*.md를 기반으로 plan mode에 진입하여 최적화 기획 문서를 작성한다.

합리적인 선에서 최대한 많은 서브에이전트를 병렬로 사용해야 한다.

**기획 결과는 청크 단위로 저장한다:**

```
step33_최적화기획_chunk1.md (500줄 이하)
step33_최적화기획_chunk2.md (500줄 이하)
step33_최적화기획_chunk3.md (500줄 이하)
...
```

**작성 규칙**:
- 각 청크는 500줄 이하로 작성 (성능 최적화)
- `.claude/hooks/research-validator.ps1`에서 각 청크 검증 (BOM/CRLF/줄수/파일크기)
- 청크 그대로 유지 (병합 안 함)

서브에이전트는 항상 haiku를 사용한다.
