**조사 단계에서 절대로 "optimize.md"를 생성하지 않는다. 오직 조사만 한다.**

**조사 단계에서 절대로 plan mode를 사용하지 않는다.**

**조사 결과는 청크 단위로 저장한다:**

```
step32_최적화조사_chunk1.md (500줄 이하)
step32_최적화조사_chunk2.md (500줄 이하)
step32_최적화조사_chunk3.md (500줄 이하)
...
```

**작성 규칙**:
- 각 청크는 500줄 이하로 작성 (성능 최적화)
