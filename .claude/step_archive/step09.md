# Step 09 - 설계

## 실행 내용

step07 기획 결과를 기반으로 plan mode에 진입하여 설계 문서를 작성한다.

**필요한 파일:**
- step08_planning_chunk*.md (기획 결과)

Class 지향으로 설계한다.

합리적인 선에서 최대한 많은 서브에이전트를 병렬로 사용해야 한다.

**설계 결과는 청크 단위로 저장한다:**

```
step09_설계_chunk1.md (500줄 이하)
step09_설계_chunk2.md (500줄 이하)
step09_설계_chunk3.md (500줄 이하)
...
```

**작성 규칙**:
- 각 청크는 500줄 이하로 작성 (성능 최적화)
- `.claude/hooks/research-validator.ps1`에서 각 청크 검증 (BOM/CRLF/줄수/파일크기)
- 청크 그대로 유지 (병합 안 함)

서브에이전트는 항상 haiku를 사용한다.
